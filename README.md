# Omniauth::Box::Oauth2

OAuth2 Strategy for Box

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-box-oauth2'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-box-oauth2
